# NUVEA-SALON-PROGRAM
Nuvea Salon adalah sistem digital untuk pendaftaran member, booking multi-layanan, dan loyalitas otomatis. Program ini meningkatkan retensi lewat level membership (Silver-Platinum) yang naik otomatis dengan fitur diskon tiap tier, notifikasi "Next Level", dan penyimpanan permanen via file .txt agar data aman.
